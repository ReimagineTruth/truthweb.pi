Setting Up Your Development Environment

Learn how to set up your local environment for TruthWeb development using Node.js, Django, and your preferred database.

  1. Clone the Repo: git clone https://github.com/username/TruthWeb.git
  2. Frontend (Node.js): Navigate to frontend/, run npm install, then npm start.
  3. Backend (Django): Navigate to backend/, create a virtual environment (python -m venv venv), activate it, and run pip install -r requirements.txt followed by python manage.py runserver.
  4. Database: Install MongoDB or PostgreSQL locally, configure in backend/truthweb/settings.py.

Using the TruthWeb API

Integrate Pi into your apps with our open-source API. Start with these steps:

  • Get an API Key: Register at the Developer Portal (coming soon).
  • Make a Request: Example using cURL: 
    curl -H "Authorization: Bearer " https://api.truthweb.com/v1/profiles/pioneer123
  • Handle Responses: Parse JSON responses in your app.
